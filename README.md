# 🌾 Raízes do Nordeste

Sistema Front-End para uma rede fictícia de lanchonetes nordestinas, desenvolvido como projeto acadêmico no curso de Análise e Desenvolvimento de Sistemas.

## 📌 Sobre o Projeto

O **Raízes do Nordeste** é uma aplicação web que simula o fluxo completo de pedidos em uma rede de lanchonetes, incluindo:

- Escolha de unidade
- Visualização de cardápio dinâmico
- Carrinho de compras
- Pagamento simulado
- Acompanhamento de pedidos
- Programa de fidelidade

O projeto foi desenvolvido com foco em **experiência do usuário, responsividade e boas práticas de desenvolvimento front-end**.

---

## 🎯 Objetivos

- Criar uma interface responsiva (Mobile First)
- Simular um fluxo real de pedidos
- Aplicar conceitos de UX/UI
- Implementar regras de negócio no front-end
- Garantir conformidade com a LGPD

---

## ⚙️ Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox + Grid + Media Queries)
- JavaScript (Vanilla JS)

---

## 🚀 Funcionalidades

### 🔐 Autenticação
- Cadastro com validação
- Login com dados mockados
- Consentimento obrigatório LGPD

### 🍽️ Cardápio Dinâmico
- Produtos por unidade (Recife, Ceará, Natal)
- Filtros por categoria
- Promoções destacadas

### 🛒 Carrinho de Compras
- Adição e remoção de itens
- Atualização em tempo real
- Total automático

### 💳 Pagamento (Simulado)
- Pix, cartão e dinheiro
- Simulação de aprovação/recusa
- Feedback ao usuário

### 📦 Status do Pedido
- Etapas:
  - Confirmado
  - Em preparo
  - Pronto
  - Retirado

### 🎁 Programa de Fidelidade
- 1 real = 1 ponto
- Resgate de recompensas
